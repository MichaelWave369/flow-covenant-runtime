# Flow Covenant Runtime v0.7 Evidence Pack

The v0.7 Evidence Pack gives Flow Covenant Runtime public examples that can be read, replayed, imported, and adapted without hidden context.

## Purpose

FCR is strongest when people can see how the model behaves in real situations. This pack turns the runtime into worked examples, fixture data, and review checklists.

## Included assets

- Worked examples by domain.
- Domain-specific example receipts.
- Lightweight JSON fixtures for future tests and import flows.
- Accessibility checklist for public practice sessions.

## Evidence boundary

An FCR evidence pack does not prove a worldview. It shows whether a scenario was handled with clearer consent, cleaner claims, visible evidence, repair paths, and traceable receipts.

## Claim-safe boundary

This is a philosophy and systems-design model, not a claim that physical laws literally want, think, or feel.
