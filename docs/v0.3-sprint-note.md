# v0.3 Sprint Note

This sprint adds a local-first scenario and receipt library.

## Scope

- Save current scenario snapshots to browser storage.
- Save generated receipts to browser storage.
- Replay saved receipts into the builder.
- Export the whole library as JSON.
- Import a library bundle or a single receipt JSON.

## Boundary

No backend, no accounts, no hidden network calls. Local library data stays in the user's browser unless exported.
