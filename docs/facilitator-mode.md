# Facilitator Mode v0.8

Facilitator Mode turns Flow Covenant Runtime into a live-room tool for meetings, community sessions, release checks, workshops, and personal reflection circles.

## Purpose

The facilitator's job is not to force consensus. The job is to keep the playfield clear enough for truth, consent, evidence, repair, and receipts to remain visible.

## Session presets

- **Reflection Session** — 25 minutes for personal or relational choices.
- **Team Review** — 45 minutes for team decisions and workflow review.
- **Community Room** — 60 minutes for public claims, promises, impacts, or trust repair.
- **Release Check** — 30 minutes for shipping, rollback, support, and evidence readiness.

## Room prompts

1. Open the room with the claim-safe boundary.
2. Name the signal before blame, urgency, or solution language.
3. Map affected parties.
4. Run the six governance gates out loud.
5. Choose the smallest reversible move.
6. Close with renewal, follow-up ownership, and a receipt.

## Closeout checks

A facilitator packet is stronger when these are visible:

- Scenario is named.
- Affected parties are visible.
- Evidence boundary exists.
- Repair path exists.
- No gate is silently blocking.
- Facilitator follow-up is named.

## Exported packet

Facilitator packets include:

- session title and mode;
- timer state;
- room prompts;
- validation checks;
- decision summary;
- gate counts;
- reversibility label;
- follow-up owner/date;
- parking lot;
- receipt;
- claim-safe boundary.
